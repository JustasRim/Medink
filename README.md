# Medink

Projekto uždavinys – palengvinti medicinos įstaigos darbuotojų darbą, pagerinant pacientų simptomų perdavimą medikams.

Pasirinktos technologijos:
- .NET 6
- React su TypeScript
- PostgreSql

Api dokumentuoti naudojamas [Swagger]

Sistema sudaro 3 modeliai:
- Medikas
- Pacientas
- Simptomas
